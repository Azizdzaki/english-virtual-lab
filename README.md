# Virtual Lab Bahasa Inggris

Virtual Lab Bahasa Inggris adalah proyek media pembelajaran interaktif untuk mata kuliah Bahasa Inggris. Dibuat dengan HTML, CSS, dan JavaScript murni, dan sepenuhnya berjalan di sisi browser (frontend).

### Prasyarat

Pastikan Anda sudah menginstal **Node.js** dan **npm**. Cara paling mudah adalah menginstalnya melalui [nvm](https://github.com/nvm-sh/nvm#installing-and-updating).

### Langkah-langkah Instalasi

1.  **Clone repositori ini :**
    ```bash
    git clone <URL_GIT_PROYEK>
    ```

2.  **Masuk ke direktori proyek :**
    ```bash
    cd <NAMA_FOLDER_PROYEK>
    ```

3.  **Install semua dependensi yang dibutuhkan :**
    ```bash
    npm install
    ```

4.  **Jalankan server pengembangan :**
    ```bash
    npm run dev
    ```
    Setelah itu buka browser dan akses alamat server lokal yang muncul di terminal Anda. Aplikasi akan otomatis mereload setiap kali Anda menyimpan perubahan pada kode.
